OpenStack

OpenStack is an open source project to build public and private clouds. Recently, I am contributing to this project. I am contributing mainly to the Release Engineering aspect of this project, helping the team to improve the way the packages are created and released.

You may refer to this link to know more about my contributions. You may find the list of patches I submitted to this project, so far.

Subversion Project

Always I like to learn new technologies, so I prefer to contribute source code to an active project. As CollabNet is one of the official sponsor of Subversion, I contributed to this project as it is very easy to know the development process. I started contributing trivial patches to this project. Long back I got partial commit access to svn2feed.py python script.

To know specific details about the contributions I have made for this project, please refer to my Subversion Contributors page.

Couple of my patches before I got partial commit access are listed below. They are sorted based on the commit date. You can visit dev and svn (commits) mailing lists for more details.

  • svn2feed.py: Place item author and date information in feed metadata
  • svnshell.py: Fix a format issue and handle exception cleanly
  • svn2feed.py: Insert "entry" tag just before the first "entry" tag
  • server-vsn.py: Support the HTTPS url
  • Update the swig bindings INSTALL document to include check-swig-py target
  • Insert the entry element as the first element in svn2feed.py
  • Atom 1.0 support for Subversion
  • Rename svn2rss.py's --url option to --item-url, and add support for feed url
  • Support ranges of revisions in svn2rss.py
  • Fix usage of header_wrappers.py
  • Remove Makefile.in targets that point to 'doc/design' directory
  • Add --max-items option for svn2rss.py
  • Add single character command line options for svn2rss.py

I work for documentation team and the team is very well focused and friendly. It is very easy to get into the team and our contributions are very well appreciated and recognized.
